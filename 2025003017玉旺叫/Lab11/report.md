# 自适应布局实验报告
## 一、实验目的
1. 掌握 Jetpack Compose 自适应布局的实现原理，实现手机与平板不同屏幕尺寸的页面适配。
2. 完成运动项目列表与详情页面开发，窄屏手机单列展示，宽屏平板双栏分栏展示。
3. 实现顶部导航栏自适应，根据屏幕模式动态控制返回按钮显示。

## 二、实验原理
利用 Compose 窗口尺寸类 `WindowSizeClass` 检测屏幕宽度，区分手机窄屏和平板宽屏：
- 手机模式：列表页与详情页分开跳转，单列布局；
- 平板模式：左侧展示列表，右侧同步展示详情，双栏布局。
通过组件复用与条件判断，实现响应式界面切换。

## 三、实验环境
- 开发工具：Android Studio
- 开发语言：Kotlin
- UI框架：Jetpack Compose
- 运行设备：手机模拟器、平板模拟器

## 四、实验步骤
1. 创建 Compose 项目，搭建基础导航结构。
2. 编写顶部导航栏组件，实现返回按钮自适应显示。
3. 实现运动列表页面与详情页面。
4. 添加屏幕宽度判断逻辑，实现平板双栏布局。
5. 分别截取手机模式、平板模式运行截图。
6. 调试代码，验证自适应布局效果。

本次实验完成了 Jetpack Compose 自适应布局开发，实现了手机和平板多设备适配。
通过判断屏幕尺寸实现布局切换，加深了对响应式界面设计的理解，提升了移动端适配开发能力。
